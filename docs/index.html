Not sure which one?

Click the Apple menu (top-left of the screen) → About This Mac. If it says Apple M1/M2/M3/M4, use the top button. If it says Intel, use the Intel button. When in doubt, try the top one first.

Install in 3 steps

  1. Open the downloaded FAB Trade file (the .dmg in your Downloads).
  2. Drag the FAB Trade icon onto the Applications folder.
  3. Open Applications, right-click FAB Trade → Open, then Open again.
The first time only, macOS may say the developer "can't be verified." That's expected for an internal app — just right-click → Open once, and it opens normally every time after.
Updates are automatic — in the app, open your profile menu (top-right) → Check for updates.
